## RPGs

### What makes a game an RPG?

Ideas:
- dialogue system
  - "animated" text box
- combat system
	- special attacks / abilities
	- enemies
- death system
	- quick saving (saving that happen
	- real saving ("save points")
- item system
	- equipment
  - single-use items (health / potions)   
	- crafting
- quest system
  - waypoints
  - puzzles?
- level design
- minimap / mapping?
